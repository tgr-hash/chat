How to Play:

- Run index.html to play the game.


Controls:

Movement: W, A, S, D or the arrow keys
Interaction: Left mouse click


Hotkeys:

Consumable items: 1, 2, 3, 4
Return to surface: B
Dismiss certain pop ups: Spacebar



IMPORTANT:

1. This game saves progress in your browser. To keep your save, always extract updates to the same folder and keep the folder name unchanged. If the path or folder name changes, the browser will treat it as a new game and your save will not appear.

2. This game uses autosave, but game data is stored in your browser. If you clear your browser cache or site data, your save will be lost. Export and back up your save regularly.
